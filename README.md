### Opening in Visual Studio Code

1. **Open the project folder:**
   - Launch Visual Studio Code
   - Click `File` > `Open Folder`
   - Navigate to the `healthy-harbour-watchers` directory and select it
   - Click `Select Folder`

2. **View the website:**
   - Right-click on `index.html` in the Explorer sidebar
   - Select "Open with Live Server" (if you have the Live Server extension installed)
   - OR simply open `index.html` in your web browser by double-clicking it

### Recommended VS Code Extensions

- **Live Server** - Launch a local development server with live reload
- **HTML CSS Support** - IntelliSense for HTML class and id attributes

### To modify content:

## ğŸ¨ Customization

### Colors
The main color scheme uses:
- Primary Blue: `#0066cc`
- Dark Blue: `#004d99`
- Green Accent: `#00cc66`

To change colors, search and replace these hex codes in `styles.css`.

### Images
To add images:
1. Create an `images` folder in the project directory
2. Place your images inside
3. Reference them in HTML: `<img src="images/your-image.jpg" alt="Description">`
